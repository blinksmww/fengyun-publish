# 剪映数字人广告脚本模板 · 卖 AI 证书

> 直接在剪映 PC 版用:**素材库 → 数字人 → 选讲师款 → 粘贴文案 → 自动出片**
> 投放方向:视频号 / 小红书 / 朋友圈广告 / 私域种子裂变
> 时长:**25–35 秒**(投流广告最优区间)

---

## 通用 4 段式结构

| 段位 | 时长 | 任务 | 写法要点 |
|---|---|---|---|
| 钩子 | 3s | 让人停下划走的手 | 反常识 / 数字 / 痛点 / 身份称呼 |
| 痛点 | 8–10s | 戳到目标用户 | 把"焦虑+具体场景"写出来 |
| 解药 | 10–15s | 证书 = 解药 | 不要列功能,讲"你能拿到什么结果" |
| 行动 | 3–5s | 闭环 | 限时 / 限量 / 福利钩 + CTA |

## 剪映操作 SOP(7 步出片)

1. 打开剪映 PC 版 → **开始创作**
2. 顶部菜单 → **数字人**
3. 形象库选**商务讲师款**(西装/职业装,匹配证书严肃感)
4. 右侧粘贴上面任一钩子 + 完整文案
5. 配音选**沉稳男声 / 知性女声**(避免可爱卡通音色,跟"考证"调性冲突)
6. 字幕样式:**底部黑底白字,加圈重点关键词**(钱、证书名、数字 用黄/红色)
7. 导出 1080P 竖屏 9:16

## 钩子写作 5 条铁律

1. **前 3 秒必须有"具体数字"或"具体身份"** — 不要"AI 时代来了"这种废话
2. **痛点要写"具体场景",不写"焦虑情绪"** — "用 Excel 做报表" > "你跟不上时代"
3. **解药段不要列课程大纲** — 用户要的是"我能拿到什么",不是"你教我什么"
4. **CTA 必须有"限制条件"** — 限时 / 限量 / 限身份(只给前 50 名),没有限制不转化
5. **每条素材只测一个变量** — 同一段广告里换钩子,中间一致,才能 A/B 出真信号
