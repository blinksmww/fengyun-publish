## 9. 已锁定的 critic / target 设计决策

[step2_report.md / critic_report.md / word_count_analysis.md 共识]

1. **target 用残差**:用 `share_residual`、`read_residual` 等(扣账号+时段+itemidx 混淆后),不直接用原始数。
2. **read_pct(账号内分位)作并行 target**:同时解决 10w+ 封顶问题,跨号可比。
3. **转发是最纯内容信号**:confounder R² 仅 0.407(最低),LGB R² 0.323(最高);critic 主信号锚 share。
4. **itemidx(头条/副条)必去混淆**:卡兹克 6.2× / 赛博禅心 21.1× / 宝玉 3.5× —— 不去混淆会主导一切。
5. **模型选 LightGBM 而非线性**:LGB R² 0.21–0.32 vs 线性 0.05–0.11,**有强非线性 + 交互**。
6. **48 维 X 特征全部作者可控**:13 标题 / 10 段落结构 / 9 标点 / 5 内容 / 4 jieba 词汇 / 5 封面色 / 2 比值。
7. **综合分权重锁定**:0.40 阅读 + 0.20 转发 + 0.15 点赞 + 0.15 在看 + 0.10 评论(微信官方算法权重)。
8. **赛博禅心待考虑独立 critic**:其字数 ρ 反向(-0.073)、lexical_diversity 反向(+0.132),与其他 3 号规律相反,可能需要号专属 critic。
9. **省略号/破折号/标题英文/问号 不进 critic 解释面板**:已证伪假相关,避免误导写稿决策。

# 深挖 2 + 2.5 双高爆款公式(2026-05-21 用户认可)

*用户原话:"什么叫爆款,能给我涨粉的,能有助于我变现的就是爆款" + "标题党的标题+深度的内容会不会更好"*  
*数据来源:reports/interaction_rate_analysis.md, title_content_quadrant.md, title_hook_formulas.md, critic_v1_report.md*

## 1. 四象限定义(已固化)

```
双高     read_pct ≥ 90 且 share_rate_pct ≥ 70    78 篇 (2.9%)
假爆款   read_pct ≥ 90 且 share_rate_pct < 30    40 篇 (1.5%)
隐藏冠军 read_pct ≤ 50 且 share_rate_pct ≥ 90    77 篇 (2.9%)
双低     read_pct ≤ 50 且 share_rate_pct ≤ 50    796 篇 (30.1%)
```

## 2. 双高文章的客观特征(对比假爆款)

| 维度 | 双高 78 篇 | 假爆款 40 篇 | 差异 |
|---|---|---|---|
| 正文字数 | **5,768 字** | 2,300 字 | 双高是 2.5 倍 |
| 平均段长 | 2,858 字 | 1,126 字 | 双高 2.5 倍 |
| 词汇多样性 | 0.357 | 0.494 | 双高用词更简单 |
| 标题字数 | 28 字 | 30 字 | 几乎一样 |
| 标题数字数 | 0.36 | **0.70** | **假爆款堆数字反而扑** |
| 「我/我们」 | 0.21 | 0.15 | 双高更个人化 |
| 封面亮度 | 181 | 119 | 双高更亮 |

**关键洞察**:**标题字数差不多但内容深度完全不同**;**数字越多反而越假爆款**(反直觉但数据明确)。

## 3. 7 大爆款标题钩子公式(LLM 从 78 篇双高提取)

```
1. 颠覆认知型     "别再用 X 了,Y 才是对的"      例:为什么我看空英伟达股票
2. 实用指南型     "爆肝 X 时,收藏这一篇就够了"   例:爆肝50小时,DeepSeek 使用技巧
3. 热点事件型     "X 发布 Y,一句话影响"          例:GPT-Image-2 全量上线
4. 人物故事型     "X 的 N 年:从起点到终点"       例:肖弘的十年狂飙
5. 思维比喻型     "技术概念 就是 生活现象"        例:Vibe Coding 是中年男人的钓鱼
6. 深度拆解型     "X 是怎么训练的|深度拆解"      例:DeepSeek-R1 是怎么训练的
7. 未来预言型     "2026 X 巨变:N 大趋势"         例:我们,已迈过奇点
```

**4 个共同特质**:
- 反常规认知(打破读者预期)
- 强行动指令(收藏/20 分钟上手)
- 高信息密度(产品名 + 数据 + 人物 + 时间)
- 情绪化表达(感叹号/emoji/反问)

## 4. 真爆款定义(用户原话锁定)

**爆款 = 能给我涨粉的、能有助于我变现的 = 微信算法权重对齐的综合分**

```
composite_pct =
   0.40 × read_pct       (阅读)
 + 0.15 × like_pct       (点赞)
 + 0.15 × old_like_pct   (在看)
 + 0.20 × share_pct      (分享)
 + 0.10 × comment_pct    (评论)
```

## 5. critic v1 模型(取代 v0)

- 模型路径:`models/critic_v1_composite.pkl`
- 目标:composite_pct(微信权重综合分)
- LightGBM CV R² = **0.326**(vs v0 share_residual 0.323,v0 read_residual 0.248)
- 单一综合模型,**适合做 writer 评分主门**

# 🔬 critic 穿透测试 — 2026-05-21(v1 vs v2 端到端对比)

**触发原因**:用户问"R²=0.42 是统计指标,但 critic 真的能区分爆款/扑街吗?"

- 测试集 = 2026 H1 全部 617 篇(训练时未见过)
- v1 train = 2026 之前全部(n=2106)
- v2 train = 2026 之前最近 12 个月(n=1383)

**穿透结果**(都翻车,但翻车方式有启发):

| 指标 | v1 | v2 | 含义 |
|---|---|---|---|
| R² | -0.421 ❌ | -0.627 ❌❌ | 比猜均值差 |
| Spearman ρ | +0.238 🟡 | +0.227 🟡 | 排序能力弱但存在 |
| Top 10% 精度 | 19.7% | 14.8% | 略好于 10% 随机 |
| Top 30% 精度 | 37.8% | **39.5%** | 略好于 30% 随机 |
| Bottom 10% 精度 | 27.9% | 24.6% | 微弱 |
| 预测 std | 7.6 | 7.8 | **真实 std 16.3** — 模型保守 |

**三个严肃发现**:
1. R² 都是负数 → critic 不能做"绝对打分"
2. ρ=+0.23 真实存在 → critic 能做"A vs B 相对比较"
3. 预测分布 std 远小于真实 → 模型不敢预测极端值

**错配案例的本质启示**:
- 过度自信(预测高但扑街):**全部赛博禅心**(跟 LOAO R²=0.28 一致,这号特殊)
- 严重低估(预测低但爆款):**Anthropic Skills 主题 + 黄仁勋热点新闻**
  - 含义:模型学不到「主题热度」+「时事热点」
  - 这是 critic 的本质局限,不是 bug

**SOP 决策(2026-05-21 锁定)**:
- ❌ 不用 critic 做绝对评分
- ✅ critic 只做「两文 A/B 相对比较」(ρ 撑得起)
- ✅ 配合 BERTopic 输出的爆款主题清单做「主题热度加权」
- ✅ 最终公式:`final_score = critic_score × topic_hotness_coef + news_bonus`
- ⏸ critic v3(加主题嵌入)推迟到深挖 5-10 完成后再做
