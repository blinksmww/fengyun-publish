## SOP 评分函数(完整规则)

评分基线 50,根据 10 个 deep dive 的方向性结论加减分:

1. 字数 3500-5500 +12 / 5500-8000 +10 / <500 -8 / >20000 -10
2. 平均段长 ≥200 +8 / ≥100 +4 / <50 -4
3. 一行一段 <10% +6 / >70% -12 / >50% -6
4. 标题字数 20-40 +3 / <10 -3
5. 标题英文字数 ≥8 +8 / <3 -3
6. 品牌词(标题):Skills +15 / Claude Code +12 / Anthropic/Claude +10 / OpenAI/GPT -5
7. 封面:白底 +6 / 暖橙 +8 / 深色 +5 / 蓝 -6 / 红 -6 / 无封面 -10
8. 图片密度 2-6/千字 +6 / >12 -5 / =0 -3
9. 词汇多样性 <0.42 +4 / >0.5 -3
10. tb_ratio <0.05 +3 / >0.15 -5
11. 致命组合(短英文标题 + 一行一段>0.5 + 短词):-15
